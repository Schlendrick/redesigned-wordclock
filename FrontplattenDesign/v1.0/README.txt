########################################
#    Aufbereitung für die Fertigung    #
########################################

Die Dateien sind für einen 50x50cm Rahmen.

Ihr findet im Archive 2 DXF Dateien. In den Dateien sind die einzelnen Bereiche des Rahmens gruppiert. Hier findet ihr nun die jeweiligen Zieltiefen für die jeweiligen Ebenen. Diese Daten benötigt ihr zum vorbereiten der Zeichnung zum fräsen auf eurer Fräse.

Bei den Zieltiefen gehe ich von einer 22mm starken MDF Platte aus.

Tiefe 0 = Plan mit Oberfläche. Tiefe -22 = komplett durchgefräst.

Datei: Wortuhr_rahmen_front_ws2812b.dxf

 - Aufhaengung aussen	: Zieltiefe -5 mm
 - Aufhaengung innen	: Zieltiefe -22 mm
 - LED			: Zieltiefe -20 mm(gefräst von vorne), oder Zieltiefe -6.5mm wenn von der Rückseite gefräst
 - Minuten LED		: Zieltiefe -20 mm(gefräst von vorne), oder Zieltiefe -6.5mm wenn von der Rückseite gefräst
 - Lichtkegel		: Zieltiefe -16.5mm
 - Lichtkegel Minuten	: Zieltiefe -16.5mm
 - Magnete Aussen 	: Zieltiefe -3 mm
 - Magnete Innen	: Zieltiefe -8 mm
 - Elektronikbox	: Zieltiefe -19 mm


Datei: Wortuhr_rahmen_back.dxf

 - LED flaeche		: Zieltiefe -2.5 mm
 - Strombuchse		: Zieltiefe -3.5 mm
 - LDR Kanal	 	: Zieltiefe -2.5 mm
 - Kabelkanal		: Zieltiefe -2.5 mm
 - Elektronik		: Zieltiefe -19 mm
